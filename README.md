# SCM 478: Operations Analysis and Modeling
### Building SCM Systems with Python, Streamlit, and GitHub

**Instructor:** John Chapman
**Semester:** Spring 2026
**Practice Company:** Peak Fuel Foods
**Capstone:** Kunzler Mega Exam

---

## Course Description

SCM 478 teaches you to build real supply chain management tools using Python, Streamlit, and GitHub. Rather than studying theory in isolation, you will construct a series of working systems for a fictional food manufacturing company — **Peak Fuel Foods** — across the semester. Each week adds a new layer of operational complexity, from product catalogs and sales dashboards to constraint-based scheduling and BOM explosion.

By the end of the course, you will have a portfolio of deployable SCM applications and the skills to extend them to any real business context. The capstone is the **Kunzler Mega Exam**, where you apply everything you have learned to a new, unseen scenario.

## Course Structure

The course runs 13 weeks and is organized around four **Transfer Exams** that assess your ability to build a working system under exam conditions:

| Weeks | Phase | Theme |
|---|---|---|
| 1–3 | Build Phase 1 | Peak Fuel dashboards and data tools |
| 4 | Transfer Exam 1 | Supply Chain Dashboard |
| 5–6 | Build Phase 2 | Forecasting methods and archiving |
| 7 | Transfer Exam 2 | Bakery Scenario |
| 8–9 | Build Phase 3 | Constraint planning and BOM explosion |
| 10 | Transfer Exam 3 | Furniture Shop |
| 11–12 | Build Phase 4 | Drum-Buffer-Rope and broken flow |
| 13 | Capstone | Kunzler Mega Exam |
